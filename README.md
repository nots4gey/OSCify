# OSCify | VRChat to Spotify Controller
OSCify is a Python program that lets you control Spotify Desktop on Windows from your VRChat avatar using it's parameters.

### Avatar Controls
 - Play / Pause <br />
 - Next Track <br />
 - Previous Track <br />
 - Spotify Volume <br />

### VRChat Parameter Setup
| Parameter Name | Parameter Type | Menu Type |
| ------------- | ------------- | ------------- |
| skipTrack  | Bool | Button  |
| previousTrack  | Bool  | Button  |
| pausePlay  | Bool  | Button  |
| spotVolume  | Float  | Radial Puppet  |
  <br />
  
Parameters can be easily added using [Avatars 3.0 Manager](https://github.com/VRLabs/Avatars-3.0-Manager) <br />
